Intro to NOS

It is important to first understand what a network is. A network is a system of links that interconnects nodes with the purpose of sharing information. Examples of this are:

  • Internet
  • Cellular
  • Transportation Networks
  • and many more...

The Internet ties together different networks via Internet Protocol (IP). 2020 provided the statistic that the internet has 4.57 billion users and 1.8 billion websites.

The Internets structure consists of a network edge (hosts, being clients and servers), Access networks and physical media (wired and wireless communication links) and a Network core (interconnected routers).

The Internet uses packet switching over circuit switching. Packet switching is when data is broken down data into smaller packets to be sent the reassembeled at its destination. Circuit switching is used by phones which is a dedicated communication path set up between two devices for a conversation. This is moreso used for traditional phone calls.
